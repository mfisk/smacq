Compiling

Flow is primarily developed and tested on Debian GNU/Linux systems, but also runs on MacOS X (Darwin) systems with Fink. The source code contains a README of the packages that must be installed to build and use the program. Unusual libraries are statically linked in so that the smacqp binary can be copied and used on most systems without requiring any libraries beyond most default system installs.
