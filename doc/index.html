Compiling

Flow is primarily developed and tested on Debian GNU/Linux systems. The source code contains a README of the Debian packages that must be installed to build and use the program. RedHat users can presumably find RPMs for those same packages. Unusual libraries are statically linked in so that the smacqp binary can be copied and used on most systems without requiring any libraries beyond default system installs.
