System for Modular Analysis and Continuous Queries

SMACQ is a...

  • Complex event processing system
  • Relational query engine for streams
  • Relational query engine for unindexed data

SMACQ is used for...

  • Data mining
  • Anomaly detection on network data

Getting SMACQ

  • [Download Directory]
    • smacq-2.7.tar.gz

  • CVS: The source code is in CVS on Sourceforge. The CVS repository is designed for developers and has many extra compilation dependencies. We recommend that users use one of the tarballs available in the download directory.

Documentation

  • smacq(1) manpage: SMACQ built-in modules
  • smacqq(1) manpage: command-line query tool using SMACQ
  • API: [Web], [PDF]
  • Paper in the 2002 SIGCOMM Internet Measurement Workshop
  • Sourceforge project web space

Mailing List

The smacq-workers@lists.sourceforge.net mailing list is used for discussions and announcements for system developers. Use Listman to subscribe.

Compiling

SMACQ is primarily developed and tested on Linux and on MacOS X (Darwin) systems with Fink. The source code contains a README of the packages that must be installed to build and use the program.
